## Cross-App Flow

Commercial product orders are decomposed using service and resource design rules. Inventory and topology provide service/resource availability, reservation, and assignment. Fulfillment executes service and resource orders, activates network/cloud/device platforms, updates inventory, and hands work to field/stock/logistics where human or physical work is required.

The original concise app overview files remain beside these folders and are linked from each app detail pack.
